EcoShop – Environment Friendly Products

EcoShop is a responsive e-commerce website showcasing environment-friendly and sustainable products.
The website allows users to browse products, add them to cart, view total prices, and proceed to buy, simulating a real online shopping experience.

Features

* Responsive Design: Works on desktop and mobile devices.
* Product Listing: View multiple eco-friendly products with images, name, and price.
* Add to Cart: Users can add products to the cart. Quantity increases automatically for repeated additions.
* Cart Page: View all items in the cart, see quantity, total per product, and grand total.
* Buy Now: Directly buy a product and navigate to checkout/cart page.
* Navigation Menu: Easy navigation between Home, Shop, About, Contact, and Cart pages.
* LocalStorage Integration: Cart items and total prices are stored locally in the browser.

Technologies Used

* HTML5 – Structure of the website
* CSS3 – Styling and layout
* JavaScript – Cart functionality, dynamic total calculation, page navigation
* GitHub Pages – Hosting the website online
* LocalStorage – Storing cart data for a persistent shopping experience
